# CT产业情报Agent · 使用指南

> 薄云咨询出品 · 2026-04

---

操作步骤

**第一步：安装与配置**

安装依赖：
- Windows：双击 `setup.bat`
- Linux/Mac：运行 `bash setup.sh`
- 或手动执行 `pip install -r requirements.txt`

配置 API Key：
1. 复制配置模板：`cp .env.example .env`
2. 编辑 `.env` 文件
3. 至少填写一个 LLM API Key（推荐 DeepSeek）：`LLM__DEEPSEEK_API_KEY`
4. 邮件/企微分发可选配置

---

**第二步：启动系统**

```bash
streamlit run web_app.py --server.port 8501
```

浏览器打开 http://localhost:8501

---

**第三步：采集情报**

1. 进入「🚀 采集中心」页面
2. 点击「🚀 开始采集」按钮
3. 实时查看采集进度（日志显示正在采集哪个源、获取多少条）
4. 等待 AI 分析完成

命令行方式：
```bash
python main.py              # 采集 + 分析 + 生成周报
python main.py --collect    # 仅采集 + AI分析
```

---

**第四步：查看情报**

进入「🔍 情报库」页面：
- 按分类/领域/重要度筛选
- 点击原文链接跳转原始网页
- 查看 AI 生成的中文摘要

---

**第五步：生成周报**

进入「📋 报告列表」页面：
- 查看历史周报
- 点击展开查看完整 Markdown/HTML 内容
- 支持邮件/企微自动分发
