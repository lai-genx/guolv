# CT产业情报Agent 使用指南

本指南用于项目解压后的首次安装、配置和启动。

## 1. 解压项目

将项目压缩包解压到本地目录，例如：

```text
D:\telecom-equipment-intel-v1.2
```

建议路径不要包含特殊符号。

## 2. 安装 Python

本项目需要 Python 3.10、3.11 或 3.12，推荐 Python 3.11。

下载地址：

```text
https://www.python.org/downloads/
```

安装时请勾选：

```text
Add python.exe to PATH
```

安装完成后，再继续下一步。

说明：项目包未内置 Python 安装包。不同电脑系统版本、权限、杀毒软件策略不同，内置安装包反而容易导致安装失败；使用官方 Python 安装包更稳定。

## 3. 安装项目依赖

双击项目根目录下的：

```text
setup.bat
```

安装脚本会自动完成：

- 创建项目虚拟环境 `.venv`
- 安装 Python 依赖
- 安装 Playwright Chromium 浏览器
- 创建运行所需目录

安装完成后，窗口会提示 `Setup finished`。

## 4. 配置 .env

打开项目根目录下的：

```text
.env
```

至少需要填写一个大模型 API Key，否则无法进行 AI 分析和周报生成。

### 4.1 DeepSeek 配置

推荐填写：

```env
LLM__DEEPSEEK_API_KEY=填写DeepSeek API Key
LLM__DEEPSEEK_BASE_URL=https://api.deepseek.com/v1
LLM__DEFAULT_MODEL=deepseek-chat
```

### 4.2 Serper 与 Jina 配置

项目默认使用 Serper 进行搜索采集和专利线索采集，必须填写：

```env
COLLECTOR__SERPER_API_KEY=填写Serper API Key
COLLECTOR__SEARCH_PROVIDER=serper
COLLECTOR__ENABLE_SEARCH=true
COLLECTOR__ENABLE_PATENT=true
```

如需增强企业官网正文读取，可选填写：

```env
COLLECTOR__JINA_API_KEY=填写Jina API Key
```

### 4.3 邮件分发配置

如需启用邮件发送，填写：

```env
DISTRIBUTION__SMTP_SERVER=填写SMTP服务器
DISTRIBUTION__SMTP_PORT=587
DISTRIBUTION__SMTP_USER=填写发件邮箱
DISTRIBUTION__SMTP_PASSWORD=填写邮箱授权码
DISTRIBUTION__EMAIL_RECIPIENTS=填写收件邮箱，多个邮箱用英文逗号分隔
DISTRIBUTION__EMAIL_SENDER=填写发件邮箱
DISTRIBUTION__ENABLE_EMAIL=true
```

不使用邮件发送时保持：

```env
DISTRIBUTION__ENABLE_EMAIL=false
```

### 4.4 企业微信分发配置

如需启用企业微信机器人，填写：

```env
DISTRIBUTION__WECHAT_WEBHOOK_URL=填写企业微信机器人Webhook地址
DISTRIBUTION__ENABLE_WECHAT=true
```

不使用企业微信发送时保持：

```env
DISTRIBUTION__ENABLE_WECHAT=false
```

## 5. 启动系统

双击项目根目录下的：

```text
startup.bat
```

也可以双击：

```text
启动文件.bat
```

启动后浏览器会打开：

```text
http://localhost:8501
```

如果浏览器没有自动打开，可手动复制上述地址访问。

## 6. 使用流程

1. 打开网页后，进入「采集中心」
2. 点击「开始采集」
3. 等待采集、AI 分析和周报生成完成
4. 到「情报库」查看情报
5. 到「报告列表」查看周报

## 7. 采集配置说明

`.env` 中以下配置可以按需要调整：

```env
COLLECTOR__ENABLE_WEB=true
COLLECTOR__WEB_MAX_SITES=0
COLLECTOR__WEB_PER_SITE_TIMEOUT=30
COLLECTOR__ENABLE_PATENT=true
COLLECTOR__PATENT_MAX_QUERIES=0
COLLECTOR__PATENT_PER_QUERY_TIMEOUT=30
```

含义：

- `COLLECTOR__WEB_MAX_SITES=0`：扫描全部企业官网
- `COLLECTOR__WEB_PER_SITE_TIMEOUT=30`：单个官网最长等待 30 秒
- `COLLECTOR__PATENT_MAX_QUERIES=0`：执行全部默认专利查询
- `COLLECTOR__PATENT_PER_QUERY_TIMEOUT=30`：单个专利查询最长等待 30 秒

如果需要缩短采集时间，可以改为：

```env
COLLECTOR__WEB_MAX_SITES=12
COLLECTOR__PATENT_MAX_QUERIES=6
```

## 8. 常见问题

### 8.1 提示找不到 Python

请重新安装 Python 3.10、3.11 或 3.12，并确认安装时勾选 `Add python.exe to PATH`。

### 8.2 启动提示 Streamlit 未安装

请先双击运行：

```text
setup.bat
```

安装完成后再启动。

### 8.3 提示没有可用 LLM

请检查 `.env` 是否填写了至少一个有效的大模型 API Key，例如：

```env
LLM__DEEPSEEK_API_KEY=填写DeepSeek API Key
```

### 8.4 Serper 搜索没有结果

请检查：

- `COLLECTOR__SERPER_API_KEY` 是否填写正确
- `COLLECTOR__SEARCH_PROVIDER=serper`
- 当前电脑是否可以访问 `https://google.serper.dev`
- Serper 账号是否还有可用额度
