## 📝 搜集指南

### 搜集方式

**方法1: 直接搜索**
```
Google: 企业名称 官方网站
Baidu: 企业名称 官网
```

**方法2: 企业信息网站**
- 企查查: https://www.qcc.com
- 天眼查: https://www.tianyancha.com
- 爱企查: https://www.aiqicha.com

**方法3: 行业目录**
- 通信世界网: https://www.cww.net.cn
- C114通信网: https://www.c114.com.cn
- 中国通信企业协会网站

### 搜集标准

**有效官网URL应该包含**：
- ✅ 有"新闻"或"新闻中心"栏目
- ✅ 企业新闻定期更新
- ✅ 使用HTTPS安全连接
- ✅ 避免重定向到第三方平台

**优先级**：
1. 企业官方网站 (最优)
2. 企业新闻中心独立域名
3. 企业官网的新闻子域名

### 格式要求

请按以下格式提供搜集结果：

```
企业名称 | 官网URL | 新闻栏目URL（可选）
example: 展讯 | https://www.unisoc.com | https://www.unisoc.com/news
```
