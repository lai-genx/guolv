## 5. 技术实现细节

### 配置文件结构
```python
{
    "name": "企业名称",
    "url": "https://企业官网/news",              # 新闻页面URL
    "list_selector": "CSS选择器列表",            # 新闻列表容器选择器
    "title_selector": "CSS选择器列表",           # 新闻标题选择器
    "link_selector": "CSS选择器列表",            # 新闻链接选择器
    "date_selector": "CSS选择器列表",            # 发布日期选择器
    "base_url": "https://企业官网",              # 基础URL（用于相对链接转换）
    "priority": "P0|P1|P2",                     # 采集优先级
    "use_jina_only": True|False                 # 是否强制使用Jina Reader API
}
```

### Fallback策略
1. **优先**: 尝试直接HTTP采集网站 (CSS选择器解析)
2. **其次**: 如果主URL失败，尝试备选URL (如有)
3. **最后**: 所有URL失败，使用Jina Reader API采集全文

新增的81个企业由于网站结构未知，默认设置 `use_jina_only=True` 确保采集成功率。
