## 采集流程中的企业识别机制

### 三层识别体系

```
第一层：关键词过滤 (collectors/base.py)
├─ 企业名称匹配（152个企业名）
├─ 技术关键词匹配（200+个关键词）
└─ 产业链词匹配（7条产业链术语）

第二���：LLM智能识别 (processors/analyzer.py)
├─ 企业名称识别
├─ 隐含企业识别（如"某产业链主要企业"）
├─ 合作方企业识别（如"与华为合作"）
└─ 自动填充 companies_mentioned 字段

第三层：RAG知识增强 (processors/rag.py)
├─ 向量相似度搜索
├─ 技术背景补充
└─ 竞争对手识别
```
