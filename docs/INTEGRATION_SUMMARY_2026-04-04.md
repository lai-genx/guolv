### 常见问题排查

**Q1: 某个网站采集为0条**
```
A: 检查日志中 use_jina_only 状态
   - 如果use_jina_only=False但采集失败
   - 查看网站CSS结构是否变化
   - 编辑web_sites_config.py中的选择器
```

**Q2: Jina API调用过多**
```
A: 这说明很多网站的直接采集失败
   - 可能是网站变更或网络问题
   - 查看日志分析具体失败原因
   - 更新CSS选择器可以减少Jina使用
```

**Q3: 想添加更多企业**
```
A: 流程如下:
   1. 获取企业官网URL
   2. 在web_sites_config.py中添加条目
   3. 设置CSS选择器（或use_jina_only=True）
   4. 重启WebCollector
```
